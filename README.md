# Parking BR 🚗

**Parking BR** é uma solução moderna e intuitiva para o gerenciamento de estacionamentos e veículos. O aplicativo foi projetado para facilitar o controle de entrada e saída, oferecendo uma interface limpa e funcionalidades que otimizam o dia a dia de operadores e proprietários.

## ✨ O que o App oferece?

- **Gestão de Veículos**: Cadastro completo de veículos com suporte aos padrões de placa Mercosul e Antiga (Cinza).
- **Controle de Tickets**: Sistema de Check-in e Check-out em tempo real com cálculo automático de permanência e custos.
- **Persistência Local**: Seus dados ficam salvos no dispositivo, permitindo o uso contínuo sem perda de informações.
- **Interface Adaptável**: Suporte total ao Modo Claro e Modo Escuro, com design baseado no Material 3.
- **Privacidade**: Tratamento seguro de dados sensíveis, como CPF mascarado e validações inteligentes.
- **Logs de Sistema**: Registro de atividades para facilitar o suporte e diagnóstico de erros.

## 🎯 Objetivo

O objetivo principal do Parking BR é demonstrar como uma aplicação mobile pode ser fluida, segura e funcional, servindo como uma ferramenta prática para o controle de pátios e garagens de forma simplificada.

## ⚠️ Status do Projeto (Alpha)

Este aplicativo está em versão **Alpha**. Ele foi desenvolvido como um projeto focado em demonstrar conceitos de design e persistência de dados. 
**Importante:** Não há foco ou planejamento para atualizações futuras, novas funcionalidades ou manutenção constante a longo prazo. O app é entregue "como está".
